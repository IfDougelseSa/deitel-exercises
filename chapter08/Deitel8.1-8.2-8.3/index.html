8.2 (Com base na Seção 8.14) Explique a noção de acesso a pacotes no Java. Explique os aspectos negativos do acesso de pacote.

Quando temos várias classes no mesmo pacote, se não definirmos algum modificador de acesso, o padrão será o acesso de pacote. Com esse modificador, quando criamos um objeto em outra classe, podemos mudar os valores dos membros que possuem o acesso de pacote na outra classe. Isto é ruim pois a lógica fica acessível para outras classes no mesmo pacote livremente.
