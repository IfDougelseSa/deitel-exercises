5.5 Descreva os quatro elementos básicos de repetição controlada por contador.

Uma variável inteira contadora, uma estrutura de repeticão, uma lógica para sair da estrutura de repetição e o um incremento para variável contadora.

5.6 Compare e contraste as instruções de repetição while e for.

Ambas podem ser utilizadas para qualquer repetição, entretanto, quando usamos a palavra chave continue na estrutura for, o programa retorna para o contador e no while volta para a condição. Além disso, o while é mais utilizado quando temos um controle por sentinela e o for por um contador definido.

5.7 Discuta uma situação em que seria mais adequado utilizar uma instrução do…while do que uma instrução while. Explique por quê.

Seria mais adequado se for necessário que a instrução seja executada pelo menos uma vez. A instrução de repetição while, se a condição for inicialmente false, o código do corpo não executará nenhuma vez. No caso do do...while, o código sempre executará pelo menos uma vez, não importando se a condição seja inicialmente false.

5.8 Compare e contraste as instruções break e continue.

A instrução break quando executada em uma instrução de repetição, sai imediatamente da mesma. A instrução continue retorna para a estrutura de repetição. Caso seja um while, retornará para o teste da condição, caso seja um for, retornará para seu incremento.
